# 无负美 Free-Beauty
while(waiting) { ∞beauty, 0borders } => everywhere

无负，是无负担。无界，是无边界。

不是又一个内容平台。
是让等待有呼吸感的地方。

巴别塔没有失败——人类敢于想象无边界，这本身就是胜利。
无负美想做的事与此同构：让审美从单一文化的中心里出走，像吹动蒲公英一样使其四散生长
图片来自世界各地，文字来自美学、哲学、社会学、文学，
上滑换一张，就这样。

# 产品逻辑
- 上滑换一张，不抢注意力
- 图片跨越地域，不局限于某一审美体系
- 每张图配一句跨学科的话
